# SQL Designer

Simple designer written in educational purposes

## How to run
```
dotnet run
```

## Shortcuts
```
ALT + P : Create New Panel
ALT + W/A/S/D : Moving

ALT + F : While editing Panel adds new field
ALT + R : While editing Panel removes last field

CTRL + E : Export to sql record
```
